# Security

RecoveryPilot AI may process sensitive recovery case information. Treat all case details, documents, identifiers, contacts, and asset data as confidential.

## Reporting

Do not open public issues for sensitive security reports. Use a private channel agreed by the project maintainers.

## Handling Sensitive Data

- Do not commit secrets, credentials, private documents, or real case data.
- Use `.env` files locally and keep `.env.example` limited to placeholder keys.
- Redact personal and financial details from logs.
- Prefer synthetic examples in tests and documentation.
